The genesis of Java

The story of Java started in 1991, at a time when Sun Microsystems thought to extend their lead in the computer workstation market into the flourishing personal electronics market. Little did anyone know that the programming language Sun was about to create would democratize computing, inspire a worldwide community, and become the platform for an enduring software development ecosystem of languages, runtime platforms, SDKs, open source projects, and lots and lots of tools. After a few years of secret development led by James Gosling, Sun released the landmark ‚Äúwrite once, run anywhere‚Äù Java platform in 1995, refocusing it beyond its original design for interactive television to applications for the betterment of World Wide Web. By the turn of the century, Java was giving spirit to everything from smartcards to space vehicles.

Today, millions of developers program in Java. Although Java continues to evolve at an ever-faster pace, on the occasion of the platform‚Äôs 25th anniversary, Java Magazine decided to take a look back at how Java molded our planet.

22. Integrated Genome Browser.
