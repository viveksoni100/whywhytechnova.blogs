What are we going to do :

Suppose there are two developers working on project named myproject. viveksoni100 (right on the below picture) is the lead developer. vivekbarbhaya (left on the below picture) is the developer. Now they want to work in collaboration.


STEP - 1 : copy the URL of the project from viveksoni100


STEP - 2 : paste the URL in the browser of vivekbarbhaya (left side)


STEP - 3 : fork the project of viveksoni100 into vivekbarbhaya



STEP - 4 : clone myproject.git in your local computer




STEP - 5 : open myfile.txt


STEP - 6 : edit myfile.txt


STEP - 7 : go in myproject directory


STEP - 8 : hit git status command


STEP - 9 : hit git add -A


STEP - 10 : hit git commit -a -m "comment"


STEP - 11 : hit git push


STEP - 12 : in vivekbarbhaya change will be reflected


STEP - 13 : create pull request in vivekbarbhaya




STEP - 14 : in viveksoni100 pull request from vivekbarbhaya will be seen


STEP - 15 : merge the pull request



STEP - 16 : change will be reflected in viveksoni100 too
